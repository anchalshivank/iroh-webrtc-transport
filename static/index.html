iroh node + WebRTC chat

WASM starts an iroh endpoint. Your node id and home relay are how others reach you over the network (N0 / relays), not a WebSocket room name.

Browser ↔ browser over iroh (QUIC signaling)

  1. Tab A: signaling iroh QUIC — wait for peer → Connect (answerer).
  2. Tab B: signaling iroh QUIC — dial peer, paste A’s node id and A’s home relay URL → Connect (offerer).
  3. Works across machines if relays/STUN can reach both; for a quick same-relay test use Same relay as me only when both tabs show the same relay.

Browser ↔ browser via WebSocket (no node ids): run signaling-server, both tabs WebSocket room, same room name, Connect.
